این پروژه یک سامانه مدیریت رویدادهای فرهنگی شهری است که با زبان ++C و به‌صورت چندفایلی پیاده‌سازی شده است.
در این سامانه امکان افزودن رویدادها، ثبت شرکت‌کنندگان و مدیریت ثبت‌نام آن‌ها با استفاده از STL Containers فراهم شده است.
برای مدیریت خطاها از Exception Handling استفاده شده تا شرایط نامعتبر مانند تکمیل ظرفیت یا شناسه نادرست کنترل شود.
کدها مطابق Standard Coding Procedures نوشته شده و توسعه پروژه با استفاده از Git و Group Coding Workflow انجام شده است.
هدف این پروژه افزایش مهارت کار تیمی، کدنویسی ساخت‌یافته و استفاده اصولی از ابزارهای استاندارد ++C است.
از جمله مشکلات پروژه میتوان به :
با تغییر دوطرف به صورت همزمان در صورت سینک نبودن دچار کانفلیکت خواهیم شد

استفاده ناهماهنگ از Reference و Copy در Containerها می‌تواند باعث ثبت نشدن تغییرات یا کپی‌های ناخواسته از اشیاء شود.

مرتب‌سازی رویدادها بر اساس Enum اگر با static_cast یا مقایسه نادرست انجام شود، ترتیب خروجی اشتباه خواهد بود.
